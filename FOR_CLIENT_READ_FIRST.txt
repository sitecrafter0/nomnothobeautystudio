## FAQ - Common Questions

**Q: Is this free?**  
A: Yes! Setting up is free. You only pay a small fee (~2-3%) when customers make real purchases.

**Q: Do they have test accounts?**  
A: Yes! All three have FREE test accounts. You can test as long as you want before going live.

**Q: Which one should I choose?**  
A: 
- If only credit cards: **Yoco**
- If want multiple options: **Payfast** 
- If want bank transfers: **Ozow**
- If want everything: **All three**

**Q: What happens if payment fails?**  
A: Customer stays on checkout page and can try again. No order created until payment succeeds.

**Q: Will my customers see the payments?**  
A: Yes. They enter the checkout, select their payment method, and pay. They get a confirmation immediately.

**Q: Can I change later?**  
A: Yes! You can add or remove payment methods anytime.

**Q: What if I'm not ready to go live?**  
A: No problem. Test as long as you want. Once you're confident, just switch to live credentials and you're ready.
