TIPS:
- Use .jpg format (you can also use .png or .webp if you prefer)
- Make sure image filenames are exactly as listed above (case-sensitive)
- Main images display at 400x400px
- Thumbnails display at 80x80px
- Use consistent aspect ratios for best appearance
